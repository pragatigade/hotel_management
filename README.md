# hotel_management
Hotel Management System: Designed and implemented a comprehensive relational database system to streamline hotel operations such as guest registration, room allocation, and billing. I began by constructing a detailed Entity-Relationship (ER) diagram to map the dependencies between guests, bookings, and services, then translated this design into a normalized SQL database. I authored complex queries using Joins and Aggregate functions to manage real-time room availability and ensure data integrity through strict primary and foreign key constraints. This project demonstrates my ability to solve real-world data redundancy issues and build efficient backend architectures for hospitality management.
